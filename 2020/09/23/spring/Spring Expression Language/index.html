Collection Selection

集合 List/Map 选择器，对于一个 List 选择器作用于其每一个元素，而 Map 作用于其 Map.Entry 。

  • 表达式中对 list 数据进行筛选 locations.?[city == 'Serbian'] 选择 city 为 Serbian 的数据。
  • locations.?[value > 10] 选择 map 中 value > 10 的数据。
